### **Option 1: Deploy as New Project (Recommended)**
1. **Create new GitHub repository** for Next.js version
2. **Push Next.js code** to new repo
3. **Connect to Vercel** as new project
4. **Use domain**: `jixiong-next.vercel.app` or similar

### **Option 2: Replace Existing Project**
1. **Replace files** in `rocky-dashboard` repo with Next.js code
2. **Update Vercel settings** to use Next.js framework
3. **Keep same domain**: `jixiong.vercel.app`

### **Option 3: Multi-Project Setup**
1. **Keep HTML version** at `jixiong.vercel.app`
2. **Deploy Next.js** to `jixiong-next.vercel.app`
3. **Test both**, then migrate

## **🚀 Quick Deployment Steps**

### **Step 1: Create GitHub Repository**
```bash
# Initialize git in Next.js project
cd /home/jixiong/.openclaw/workspace/jixiong-website
git init
git add .
git commit -m "Initial Next.js version of personal website"
```

### **Step 2: Create New GitHub Repo**
1. Go to https://github.com/new
2. Name: `jixiong-website` (or similar)
3. Don't initialize with README
4. Create repository

### **Step 3: Push to GitHub**
```bash
git remote add origin https://github.com/arthanusamn/jixiong-website.git
git branch -M main
git push -u origin main
```

### **Step 4: Deploy to Vercel**
1. Go to https://vercel.com/new
2. Import `jixiong-website` repository
3. Configure:
   - **Framework Preset**: Next.js
   - **Build Command**: `npm run build`
   - **Output Directory**: `.next`
   - **Install Command**: `npm install`
4. Click **Deploy**

## **🚨 Troubleshooting**

### **Build Errors**
```bash
# Clean install
rm -rf node_modules package-lock.json
npm install
npm run build
```

### **Deployment Issues**
1. Check Vercel logs
2. Verify environment variables
3. Ensure Node.js version compatibility
4. Check file permissions

### **Domain Issues**
1. Verify DNS propagation (24-48 hours)
2. Check SSL certificate (auto-generated by Vercel)
3. Clear browser cache
